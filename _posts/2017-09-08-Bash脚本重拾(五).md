#### 读取文件所有内容

掌握文件的输入重定向

```
cat filename
```

或

```
#!/bin/bash
cat jenkins问题记录.txt | while read line 
do
	echo $line
done 
```

或

```
#!/bin/bash
while read line 
do
	echo $line
done < jenkins问题记录.txt
```



#### 创建个文件

`while`的用法

```
#!/bin/bash
while :				# 注意写法 : 前面有一个空格
do
	echo "please input file's name:"
	read a
	if test -e $a
		then
		echo "the file is existing Please input new file name"
	else
		touch $a
		echo "you are successful"
		break
	fi
done
```



#### 查询本机地址

管道用法，流编辑，纵向编辑

```
ifconfig | grep "inet" | awk '{print $2}' | sed -n '5p'
```
